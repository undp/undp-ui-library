# UNDP design system for react ![npm](https://img.shields.io/npm/v/@undp/design-system-react)

##### Here you can find our design guidelines, component documentation, and resources for building a react or next.js apps with UNDP's design system. It is base on the design system build and maintained by UNDP's Digital Communication Studio. This resource is build and maintained by UNDP's BPPS team.

UNDP design system react is the implementation of [UNDP design system](https://design.undp.org/) in React. Before starting, it is recommended to learn React first, and correctly install and configure Node.js v18 or above.

We also assumes that you have intermediate knowledge about HTML, CSS, and JavaScript/TypeScript, and React. Starting out is straightforward.

<br />

## Getting started

#### Using the library in React projects

Before starting, it is recommended to correctly install and configure [Node.js](https://nodejs.org/en/download) v18 or above.

We also assumes that you have intermediate knowledge about HTML, CSS, and JavaScript/TypeScript, and React. Starting out is straightforward.

#### Installing with NPM

Start by installing UNDP Design System as a node module and save it as a dependency in your package.json:

```
npm install @undp/design-system-react `--save`
```

#### Import

It is recommended to import what you need and the use it.

It is also recommended to import the css because some setting expect the CSS to be imported to look good. You can import the css file like this:

```
import '@undp/design-system-react/style.css';
```

## Adding custom color

> To add custom color as the primary color overwrite the following variable in main css and import it after the css for the design system: `--custom-color-100`, `--custom-color-200`, `--custom-color-300`, `--custom-color-400`, `--custom-color-500`, `--custom-color-600`, `--custom-color-700`, `--custom-foreground`. For example:

```html
<div
  style={{
    '--custom-color-100': '#FFE5B4',
    '--custom-color-200': '#FFD38C',
    '--custom-color-300': '#FFB85C',
    '--custom-color-400': '#FFA242',
    '--custom-color-500': '#FF971D',
    '--custom-color-600': '#FD9D24',
    '--custom-color-700': '#E6851D',
    '--custom-foreground': '#fff'
    }}
>
<!--HTML element goes here-->
</div>

```

After that use the color variant as `custom` in various component that allows it. For example:

```tsx
<Container backgroundColor='custom' gap='none' layout='default' padding='base' width='full'>
  <H4>Heading</H4>
  <P>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec blandit augue eu sagittis
    facilisis. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos
    himenaeos.
  </P>
</Container>
```

## Using the design system with TailwindCSS

UNDP Design System is build using TailwindCSS. If you want to use the design system in a project with TailwindCSS, then to avoid conflicts please use the tailwind config from the design system as preset in you tailwind.config.js

```js
// tailwind.config.js
const preset = require('@undp/design-system-react/tailwind.config');

export default {
  presets: [preset],
  ...
};
```

## TypeScript

UNDP Design System provides a built-in ts definition, you don't need to install any type definitions.
